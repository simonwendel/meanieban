Welcome to MeanieBan!

MeanieBan is an AngularJS-based clone of a classic japanese game called Sokoban, which translates to "warehouse keeper".

The object of the game is to move the keeper to push the boxes until they cover all targets in the level. Trying to do it in as few moves as possible can be quite challenging and finding the optimal solution can be a daunting task.

Fine print for developers

The game was developed with the goal of using some new-found knowledge of AngularJS. Although the name does imply it, the game is not backed by the technologies that make up the rest of the MEAN acronym. The level data was imported into a AngularJS value object by a Node.js script, but that is as far as it goes.

It has been a learning experience, both with finding out how AngularJS apps work and what tools are out there to aid in their development. You might find some bugs, I am working on fixing a few, but please use the email link in the footer to let me know if you find something.

If you're interested in the source code you can clone it or fork it on Github. The game contains content that was not created by me, please see the attribution page for details.
